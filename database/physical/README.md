# Physical Database Model

This directory contains the **physical database model**.

Here, the logical model is converted into a **real implementation**, considering the chosen DBMS and technical constraints.

## What you will find here
- SQL scripts

## Dependencies
- Specific DBMS (e.g., MySQL, PostgreSQL, etc.)

## Purpose of this directory
To practice implementing a real database from the logical model, reinforcing learning with SQL and DBMS concepts.
